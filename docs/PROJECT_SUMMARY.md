# Daymon 프로젝트 요약

## 게임 컨셉

- **걸음수로 키우는 AI 몬스터 육성 게임**
- **"영혼 + 껍데기" 시스템**
  - AI 영혼: 영원히 유지
  - 몬스터(껍데기): 진화/교배
- 영혼은 유저를 기억하고 배워가는 **"디지털 동반자"**  
  (진짜 AI 친구가 아닌, 기억해주는 동반자)

---

## 속성 시스템

| 요일 | 속성 |
|------|------|
| 월 | 달 |
| 화 | 불 |
| 수 | 물 |
| 목 | 나무 |
| 금 | 금속 |
| 토 | 땅 |
| 일 | 태양 |

- 현실 요일과 연동 (예: 화요일 접속 시 화 속성 보너스)
- 속성은 **주로 외형**에 영향

---

## 몬스터

- **시작**: 원(구) 형태의 단순한 정령
- 시간/걸음수에 따라 **형태 변화**
- **교배**로 속성 조합 → 나만의 캐릭터
- 나중에 IP 콜라보 가능하도록 **스킨 시스템** 고려

---

## 핵심 가치

- **현타 방지**: 영혼은 절대 사라지지 않음, 추억 쌓임
- **건강한 과금**: Pay to Win 아님, 편의성/스킨만
- **장기 플레이**: 교배 주기로 게임 수명 조절

---

## MVP (지금 만들 것)

- **메인 화면**: 몬스터(원형 정령) + 걸음수 입력 + 대화 버튼
- **영혼 시스템**: Claude API로 대화 + 기억
- **걸음수 → 성장**: 웹에서는 수동 입력, 나중에 앱으로 자동 연동
- **간단한 진화**
