## 1. 재화 시스템

### 골드 🪙 (인게임 통화)
- **획득**: 알 부화, 일일 출석, 몬스터 방출, 안식처 수익, 탐험 보상
- **소비**: 알 구매, 부화 가속(약간), 슬롯 해금, 안식처 확장, 교배 비용

### 다이아 💎 (프리미엄 통화, 나중에)
- **획득**: 앱 결제, 업적 보상, 이벤트
- **소비**: 편의 기능(시간 단축 20~30%), 꾸미기, 프리미엄 알 뽑기
- ❌ 몬스터 직접 구매 불가

### 과금 철학
- "Pay for Convenience, Not Power"
- 무료 유저 = 시간 투자 → 같은 결과 도달
- 유료 유저 = 시간 절약 + 꾸미기
- 무료 유저가 게임을 받쳐주고, 소수 유료 유저에서 수익

## 2. 몬스터 시스템

### 기본 형태
- 15~20종의 기본 몬스터 형태 (완성형 디자인)
- "현실 동물 모티브의 판타지 생물" (고양이→미스트캣, 늑대→블레이즈울프)
- 각 형태 × 7속성 = 개별 AI 생성 스프라이트 (CSS 필터 X, 퀄리티 중요)

### 속성 (7종)
- 🔥 불 / 💧 물 / 🌿 나무 / ⚡ 금속 / 🪨 땅 / ☀️ 빛 / 🌙 어둠

### 생사
- ❌ 몬스터는 죽지 않음
- 방치 시 😴 잠듦 → 먹이 주면 깨어남 (가벼운 재접속 유도)

### 성장
- 알 → 아기 → 성체 → 최종 진화
- 속성별 진화 분기 가능

### 세대 시스템 ⭐
- 1세대 (알에서 바로 부화): 경험치 ×1.0
- 2세대 (1세대끼리 교배): 경험치 ×1.15 (+15%)
- 3세대: ×1.25, 4세대: ×1.35, 5세대: ×1.40
- 최대 10세대: ×1.50 (상한선, 수확 체감)
- 세대가 높을수록 돌연변이 확률도 증가
- 목적: 교배 동기, 성장 부담 감소, 오래 할수록 이점

### 스탯 방향
- ❌ PvP 전투력 없음
- ✅ PvE 활용 스탯 (탐험/던전 보상 효율에 영향)
- 속성 상성이 핵심 전략

### 용도
- 수집 (도감 완성)
- 탐험/요일 던전 (방치형, 보상 획득)
- 교배 (새 알 생성, 세대 업)
- 안식처 배치 (꾸미기)

## 3. 교배/유전 시스템 ⭐ 핵심

### 유전자 구조
```
몬스터 = {
  species: "cat",          // 형태 (부모 중 하나 상속)
  element: "fire",         // 속성 (부모 중 하나 or 혼합)
  genes: {
    color: "orange",       // 색 유전자
    pattern: "stripes",    // 무늬 유전자
    size: "medium",        // 크기 유전자
    personality: "brave",  // 성격 유전자
    eyeColor: "red",       // 눈 색 유전자
    special: "rainbow",    // 특수 유전자 (희귀)
  }
}
```

### 교배 규칙
- 형태: 부모 중 랜덤 상속 (특수 조합은 전용 형태)
- 속성: 부모 중 랜덤 (희귀 확률로 혼합 속성)
- 유전자: 각 유전자마다 부모 중 랜덤 + 소량 돌연변이

### ⭐ 소셜 교배 (친구와 교배)
- 친구코드로 연결 → 서로의 몬스터로 교배 가능
- "나한텐 없는 무지개 유전자를 친구에게서 받을 수 있음"
- 교배의 소셜 동기 부여

### 유전 다양성 유지 (수렴 방지)
- **돌연변이**: 교배 시 소확률로 새로운 유전자 등장
- **계절 유전자**: 특정 시즌에만 나타나는 유전자 (봄: 벚꽃, 여름: 해변)
- **신규 유저 보너스**: 가입 시 랜덤 시드로 고유 유전자 부여
- **열성 유전자**: 숨어있다가 특정 조합에서만 발현
- **세대 보너스**: 교배 세대가 높을수록 돌연변이 확률 증가

## 4. 요일 던전 시스템 ("데이몬" = Day + Monster)

### 요일별 속성 던전
| 요일 | 속성 | 던전 이름 (예시) |
|---|---|---|
| 월(月) | 🌙 어둠 | 달빛 미궁 |
| 화(火) | 🔥 불 | 화염 동굴 |
| 수(水) | 💧 물 | 심해 유적 |
| 목(木) | 🌿 나무 | 고대 숲 |
| 금(金) | ⚡ 금속 | 기계 공장 |
| 토(土) | 🪨 땅 | 수정 광산 |
| 일(日) | ☀️ 빛 | 천공 신전 |

### 규칙
- 일일 1회 참여 (매일 접속 동기)
- 상성 유리한 속성 → 3층까지 클리어 가능
- 상성 불리 → 1~2층에서 실패
- 층별 보상 증가 (골드, 알 조각, 전용 아이템)
- 각 던전에서만 드롭되는 희귀 아이템 존재

### 팀 편성 (나중에)
- 최대 3마리 파티
- 속성 조합 보너스

## 5. 소셜 시스템

### 과시 채널 (우선순위)
1. 프로필 카드 공유 (SNS)
2. 도감 (컬렉션)
3. 안식처 방문 (친구)
4. 랭킹 (도감 완성률, 부화 횟수 기반 - 전투력 X)

### 친구 시스템
- 친구코드로 연결
- 소셜 교배
- 안식처 방문
